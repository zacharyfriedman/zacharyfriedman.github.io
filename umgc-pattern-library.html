University of Maryland Global Campus - Pattern Library

Product:
UMGC Classroom - Pattern Library
Sponsor:
University of Maryland Global Campus
Role:
User Experience Designer

The aim of this project was to create a pattern library to improve the learner experience in the UMGC classroom. The pattern library serves as a comprehensive resource for faculty, learning designers, technologists, and other UMGC staff, providing them with a collection of pre-designed and accessible user interface elements and components, such as typographic styles, callout boxes, cards, and other common design patterns used in higher education courses.

One goal of this project was to provide a consistent user experience across all UMGC courses. By creating a pattern library, it was easier for faculty to design and build courses that are easy to use and navigate, regardless of the learnersâ€™ familiarity with the course content. With a well-organized and accessible pattern library, UMGC also saves time and money, as learning designers and technologists can quickly and easily access pre-designed components and configure them to fit the needs of their learning objectives. Ultimately, the pattern library improved the user experience of courses, making them more accessible and user-friendly for students, and improving the authoring experience for faculty and staff.
